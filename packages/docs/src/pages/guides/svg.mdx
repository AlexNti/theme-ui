# Using SVG

You can use the `sx` prop directly on SVG elements to add theme-based styles.
The SVG `fill` and `stroke` properties will pick up values from `theme.colors`, but you can also take advantage of the `currentcolor` keyword to inherit color from parent HTML elements.
This can be handy when working with [color modes](/color-modes) and when using single color SVG elements, like icons.

```jsx
/** @jsx jsx */
import { jsx } from 'theme-ui'

// example hamburger icon using current color
export default ({ size, ...props }) =>
  <svg
    xmlns="http://www.w3.org/2000/svg"
    width={size}
    height={size}
    fill="currentcolor"
    viewBox="0 0 24 24"
    sx={{
      display: 'block',
      margin: 0,
    }}>
    <path d="M3 18h18v-2H3v2zm0-5h18v-2H3v2zm0-7v2h18V6H3z" />
  </svg>
```

```jsx live=true
// example with colors
<svg
  xmlns="http://www.w3.org/2000/svg"
  viewBox='0 0 32 32'
  width='96'
  height='96'
  sx={{
    fill: 'primary',
    stroke: 'secondary',
    strokeWidth: 2,
  }}>
  <circle
    cx={16}
    cy={16}
    r={12}
  />
</svg>
```
