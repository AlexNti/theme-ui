# Style Guide

Theme UI includes an optional package for building style guides based on the [Theme Spec](/theme-spec).

```sh
npm i @theme-ui/style-guide
```

```jsx
<ThemeProvider theme={theme}>
  <ColorPalette />
  <TypeScale />
  <TypeStyle
    fontFamily='monospace'
  />
</ThemeProvider>
```
