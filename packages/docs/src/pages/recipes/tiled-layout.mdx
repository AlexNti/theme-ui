# Tiled Layout

Create a responsive auto-sized tiled layout using CSS Grid.

```jsx
/** @jsx jsx */
import { jsx } from 'theme-ui'

export default ({
  gap = 3,
  width = 128,
  ...props
}) =>
  <div
    {...props}
    sx={{
      display: 'grid',
      gridGap: gap,
      gridTemplateColumns: `repeat(auto-fit, minmax(${width}px, 1fr))`,
    }}
  />
```
