# Custom Pragma

To avoid manually importing and calling the `css` utility, a custom pragma can be used instead.
Under the hood, this custom pragma converts a Styled System-aware `sx` prop into a style object and passes it to the Emotion `jsx` pragma.
This is intended as a complete replacement for the Emotion custom pragma.

```jsx
/** @jsx jsx */
import { jsx } from 'theme-ui'

export default props => (
  <div
    {...props}
    sx={{
      color: 'white',
      bg: 'primary',
    }}
  />
)
```

To opt-out of using Styled System CSS, use the `css` prop to render raw CSS values.

```jsx
/** @jsx jsx */
import { jsx } from 'theme-ui'

export default props => (
  <div
    {...props}
    css={{
      // raw values
      color: 'tomato',
      fontSize: 24,
    }}
  />
)
```
