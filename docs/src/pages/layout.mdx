# Layout Components

```js
import {
  Layout,
  Header,
  Main,
  Container,
  Footer
} from 'theme-ui'
```

```jsx
// example composition
<Layout>
  <Header />
  <Main>
    <Sidebar />
    <Container>
      {props.children}
    </Container>
  </Main>
  <Footer />
</Layout>
```

- `Layout` ~~`Page`~~ (name TBD) the root styled component for wrapping other layout components
- `Header` section for top of the page
- `Main` the center of the page
- `Container` a max-width, centered area
- `Footer` section for the bottom of the page
- `Sidebar` (not yet implemented) a sidebar with, ideally, a mechanism for responsive layouts

## Box

A lower-level `Box` primitive component is included as well.
This is similar to the [Rebass Box](https://rebassjs.org/Box) component, but built with Emotion and `@styled-system/css`

```js
import { Box } from 'theme-ui'
```

## Layout Theming

These page layout components can be styled with the `theme.styles` object, similar to how other MDX components.

```js
theme: {
  styles: {
    Header: {
      color: 'white',
      bg: 'black',
    }
  }
}
```

The styles for the layout components can reference colors and other theme values using `@styled-system/css`. This can be used to theme specific aspects like colors independent from the component.

```js
theme: {
  colors: {
    header: {
      text: '#fff',
      background: '#000',
    },
  },
  styles: {
    Header: {
      // references values in `theme.colors`
      color: 'header.text',
      bg: 'header.background',
    },
  },
}
```
